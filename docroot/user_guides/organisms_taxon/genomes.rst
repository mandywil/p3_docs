Genomes Tab
============
The Genomes Tab provides a table listing of the genomes in the selected Taxon View, as described in the link below. Genomes in PATRIC include extensive additional information including annotations, metadata, and where available, antimicrobial resisistance (AMR) data, each described in detail from the links below.
